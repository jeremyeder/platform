# Hello ACP Labs

Welcome to the Hello ACP lab series! These five quick labs will introduce you to the Ambient Code Platform's coolest capabilities. Each lab takes just 2-3 minutes and is designed to be fun, foolproof, and eye-opening.

## What You'll Learn

The Hello ACP series showcases different aspects of the platform:

1. **Session Creation** - See AI execute tasks automatically
2. **Code Generation** - Watch AI write actual code
3. **Multi-Repo Operations** - Work across multiple repositories simultaneously
4. **RFE Workflow** - Experience the 7-agent engineering council
5. **Interactive Mode** - Chat with AI in real-time

## Time Commitment

- **Per Lab**: 2-3 minutes
- **Full Series**: 10-15 minutes
- **Total Fun**: Guaranteed!

## Prerequisites

- Access to the Ambient Code Platform UI
- A project already created (or permission to create one)
- That's it! No coding required.

## The Labs

### [Lab 1: Hello ACP - Your First Session](01-first-session.md)

**Time**: 2 minutes | **Difficulty**: Easiest

Create your first AI session and watch it create a "Hello ACP" file. Perfect introduction to how ACP executes tasks.

**You'll Learn**: Session creation, viewing results, understanding execution flow

---

### [Lab 2: Hello Code - AI That Writes Code](02-hello-code.md)

**Time**: 3 minutes | **Difficulty**: Easy

Watch the AI write a complete "Hello ACP" program in Python. See code generation in action.

**You'll Learn**: Code generation, file creation, reviewing AI-written code

---

### [Lab 3: Hello Multi-Repo](03-multi-repo.md)

**Time**: 3 minutes | **Difficulty**: Easy

Create the same file in two different repositories simultaneously. Experience ACP's multi-repo superpower.

**You'll Learn**: Multi-repository operations, workspace management, synchronized changes

---

### [Lab 4: Hello RFE - The 7-Agent Council](04-hello-rfe.md)

**Time**: 3 minutes | **Difficulty**: Medium

Submit a simple Request for Enhancement and watch 7 specialized agents collaborate to refine it.

**You'll Learn**: RFE workflow, agent collaboration, engineering council pattern

---

### [Lab 5: Hello Interactive - Chat With Your AI](05-interactive.md)

**Time**: 2 minutes | **Difficulty**: Easy

Have a real-time conversation with an AI agent. Ask questions, get answers, iterate together.

**You'll Learn**: Interactive mode, real-time collaboration, conversational AI

---

## Learning Path

**New to ACP?** Start with Lab 1 and work through in order. Each lab builds on concepts from the previous one.

**Want something specific?**

- Interested in code generation? → Lab 2
- Need to work across repos? → Lab 3
- Want to see agent collaboration? → Lab 4
- Prefer interactive work? → Lab 5
